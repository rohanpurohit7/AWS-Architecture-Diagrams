# Learning Path

## Foundation
- AWS account structure and landing zones
- IAM and identity federation
- VPC, subnets, routes, NACLs, security groups
- NAT, internet gateways, endpoints, DNS

## Intermediate
- ALB/NLB, Auto Scaling, ECS, EKS, Lambda
- RDS/Aurora, DynamoDB, S3, ElastiCache
- Kinesis, MSK, Glue, Athena, OpenSearch
- CloudWatch, Prometheus, Grafana, X-Ray

## Advanced
- multi-region active/active and DR
- Transit Gateway and centralized inspection
- AWS Network Firewall and NGFW
- Security Lake, Security Hub, Detective
- Bedrock, RAG, guardrails, agents, MCP
- KNN and ensemble anomaly detection
- SOAR and human-approved containment
